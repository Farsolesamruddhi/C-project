# C-project

Movie Ticket Cost Calculator (C Project)

This is a simple C programming project that calculates the movie ticket cost based on:

- Movie Category (2D / 3D)
- Circle Type (gold / silver)

The program takes user inputs such as movie name, duration, category, and seating circle, then validates them and displays the correct ticket cost.
It also handles different invalid input scenarios and returns proper error codes.

 Features:- 
 Get movie details from user
 Validate movie category (2D/3D)
 Validate circle type (gold/silver)
 Calculate ticket price
 Return error codes for invalid inputs
 Display proper messages to the user
